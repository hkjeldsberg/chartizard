## Notes

- **Deps likely needed:** none. UML shapes are pure SVG paths (rects, lines, arrowheads via `<marker>`). FSM + PERT reuse the network-diagram pattern (positions hand-rolled). Manhattan plot is a scatter with alternating-colour x-bands. H-R diagram is a log-scale scatter with reversed x-axis. Climate stripes is 170 colored `<rect>` elements.
- **Known trickiness:**
  - **UML arrows** need SVG `<marker>` elements for the four arrowhead shapes (open-triangle inheritance, filled-diamond composition, hollow-diamond aggregation, open-arrow association). Define them once in `<defs>` and refer by `markerEnd={"url(#inheritance)"}`.
  - **Crow's-foot notation** on the ERD needs marker assets for each cardinality (0..1, 1, 0..*, 1..*). At 120×80 thumbnail scale these get lost — let the thumbnail just show entity boxes with a relationship line, no cardinality glyphs. The live chart carries the real encoding.
  - **Sequence diagram** activation bars are thin vertical rectangles overlaying lifelines between sync-call and its return. Message arrows connect the activation rectangles, not the lifelines directly.
  - **FSM** needs clear initial-state (● + arrow) and accepting-state (◎ double-circle) conventions. Self-loops need curved arrow paths (not just arc of radius r around a node point — start/end tangents must differ).
  - **Kanban board** WIP limits + card count per column need to be pre-placed in the dataset, not computed live. Cards positioned with fixed y-offsets within each column.
  - **PERT critical path** — pre-compute the critical path in the dataset, don't run the forward/backward pass at render. One hand-picked zero-slack chain highlighted red.
  - **Climate stripes** values are synthetic (seeded LCG approximating HadCRUT5 anomaly trajectory: slight cooling 1850-1910, slow warming 1910-1980, rapid warming 1980-present). Do not fetch real data.
  - **Magic Quadrant** bubble placement: ~12 named vendors with hand-chosen (x, y) values that land across all 4 quadrants. Use realistic-sounding but fictional names to dodge trademark silliness.
  - **Manhattan plot** — 22 chromosome bands. Total points ~3000. Use a seeded LCG; spike the p-values at ~6 known loci on chromosomes 3, 7, 9, 11, 17, 19. Alternate grey/light-grey for the chromosome bands to read the skyline.
  - **H-R diagram** — X-axis goes hot→cold (O-type stars on LEFT). Use `scaleLog` domain reversed (high to low temp) or a cleverer ordinal mapping. Y-axis is log luminosity, with the Sun at y=0. Cluster main-sequence stars along the diagonal; scatter red-giants top-right, white-dwarfs bottom-left.
- **Slot routing:**
  - **Wave A** — all Sonnet. These are small charts with self-contained layouts.
  - **Wave B** — Opus for B1 (UML class + ERD: N>6 distinct shape primitives with distinct arrow semantics, each arrow type a different marker), B2 (sequence + FSM: N=8+ messaging geometry + state-transition geometry), B3 (kanban + PERT: column layout + critical-path layout). Sonnet for B4 (climate-stripes + magic-quadrant: both essentially scatter-with-structure, no complex coordination) and B5 (manhattan + H-R: both scatter-based with specialty axes, N<6 distinct component types).
- **Tile-size bumps anticipated:** none. Placeholder sizes look right.
