## Notes

- **Deps likely needed:** none. All charts compose from existing SVG primitives + `d3-shape` + `d3-array`. No new layout libraries.
- **Known trickiness:**
  - **Chernoff faces** ‚Äî 6 variables ‚Üí 6 facial features. Use a parametric face drawing: face ellipse (width = x‚ÇÅ), two eye ellipses (radius = x‚ÇÇ, separation = x‚ÇÉ), nose (length = x‚ÇÑ), mouth (curvature = x‚ÇÖ via cubic Bezier, width = x‚ÇÜ). Normalise each variable to [0.3, 1.0] so faces don't go degenerate. Label each face with its observation id.
  - **Andrews plot** ‚Äî f(t) as a parametric curve in (t, f_x(t)). Sample 60 t-values in [‚àíœÄ, œÄ], evaluate f for each observation. Plot all curves with low opacity; thicker/coloured by class label (e.g. Iris species).
  - **Star glyph** ‚Äî polygon with N vertices at (r_i ¬∑ cos(Œ∏_i), r_i ¬∑ sin(Œ∏_i)) where Œ∏_i = 2œÄi/N. Draw in small-multiples grid: position each glyph at (col ¬∑ cellW + cellW/2, row ¬∑ cellH + cellH/2); glyph radius = min(cellW, cellH)/2 ‚àí 4px.
  - **Hammock plot** ‚Äî categorical parallel-coords. 4 axes evenly spaced horizontally. Between axes i and (i+1): for each (category at i, category at i+1) pair, draw a trapezoid whose width at axis i = count(pair) / total, narrowing to width at axis (i+1) = count(pair) / total. Stack trapezoids vertically at each axis.
  - **Doubledecker** ‚Äî subdivide x-axis into predictor-category combinations (widths ‚àù joint counts). For each vertical bar, shade bottom fraction = response rate within that cell; top fraction = non-response.
  - **Barcode chart** ‚Äî single horizontal baseline; vertical ticks at each observation's value; tick length encodes a secondary variable (e.g. earthquake magnitude).
  - **Root locus** ‚Äî Evans's 8 rules. For G(s) = 1/((s+1)(s+2)(s+3)): 3 branches, centroid = (-1-2-3)/3 = -2, asymptote angles = ¬±60¬∞ and 180¬∞. Hand-compute ~30 points per branch at increasing K; plot as smooth curves. Mark the imaginary-axis crossing (where K makes system marginally stable ‚Äî Routh-Hurwitz gives K_crit = 60 for this example).
  - **Periodogram** ‚Äî compute DFT of the synthetic signal: |X(f_k)|¬≤ at f_k = k/N, k = 0..N/2. Plot as a stem-and-circle or bar chart with stems at each f_k; peaks should stand out at f = 1/12 and f = 1/6.
  - **Tissot indicatrix** ‚Äî on a Mercator projection show tiny circles at (Œª, œÜ) grid. On Mercator, a circle of radius r on the sphere at latitude œÜ projects to an ellipse with horizontal radius r¬∑sec(œÜ), vertical radius r¬∑sec(œÜ) ‚Äî isotropic but growing with latitude (that's why Greenland bloats).
  - **Isochrone map** ‚Äî simplified city outline + 4-5 irregular contour polygons at 10/20/30/45 min. Don't worry about real routing; hand-draw plausible shapes (tighter to highways, pinched at rivers).
  - **Argument map** ‚Äî tree with 3 supporting + 2 opposing branches. Nodes as rounded-rectangles. Green fill/border for support, red for oppose. Arrows from sub-reason up to reason, reason up to main claim.
  - **Rich picture diagram** ‚Äî deliberately hand-sketched appearance. Use `stroke="currentColor"` with jittered path data to suggest pencil lines. Stick figures = 4-line torso + circle-head. Thought bubbles = scalloped ellipse. Crossed swords = two rotated rectangles with pommel.
  - **Bubble map** ‚Äî one central circle + 6-7 satellite circles arranged on a ring around it. Straight lines from centre to each satellite.
  - **Double bubble map** ‚Äî two central circles offset by ~40% of width. Shared attributes in 3 circles centred between; unique attributes in 3 circles on outer side of each central. Lines from each central to its connected attributes.
  - **Surface plot** ‚Äî isometric projection: x-axis ‚Üí (+cos30¬∞, ‚àísin30¬∞), y-axis ‚Üí (-cos30¬∞, -sin30¬∞), z-axis ‚Üí (0, 1). Compute sombrero f(x,y) on 25√ó25 grid. Draw back-to-front: for each row from far to near, draw the x-curve at that y. Hidden-line removal = natural back-to-front painter's algorithm.
  - **Logarithmic plot** ‚Äî axis with log ticks at 10, 20, 50, 100, 200, 500, 1000. For Zipf: rank 1..100 on log x; frequency values on log y; near-straight line with slope ~-1.
  - **Pictorial percentage** ‚Äî 10√ó10 grid of stick-person icons. Icon = small path (5 line-segments for torso+limbs, one circle for head). Coloured icons for the percentage fraction; grey icons for the remainder.
  - **Unit chart** ‚Äî one mark per unit, laid out in rows of ~20 marks or packed hexagonally. For counts ‚â§100 use a 10-wide grid; for higher counts use a denser packing.
  - **Genogram** ‚Äî 3 rows (grandparents, parents, children). Shapes: 20px rects for squares, 20px circles for circles. Horizontal "married" lines between couples; vertical "descent" lines from marriage-middle down to child-row. Shaded shapes for the trait. Ages/dates next to key figures.
