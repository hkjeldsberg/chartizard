## Notes

- **Deps likely needed:** none. Voronoi can be hand-rolled from Lloyd's relaxation over a ~50×50 grid (approximate, good enough for 15 seeds) OR — if `d3-delaunay` is already installed — used directly. Check `node_modules/d3-delaunay` first; if absent, flag (don't install). Every other chart uses the existing Visx + d3-shape / d3-hierarchy / d3-array stack.
- **Known trickiness:**
  - **Nightingale's equal-area vs Polar-area's equal-angle** — these are distinct conventions. Nightingale: each wedge has the same *angular width*, radius encoding the value → area scales linearly with value (what readers actually perceive). Polar-area: each wedge is sized by *angle*, radius constant → area encodes value. Be precise about which is which.
  - **Voronoi** — if `d3-delaunay` isn't available, use a discrete "assign each pixel to nearest seed" approximation rendered as coloured rects, or fall back to straight-line perpendicular-bisector construction for 15 fixed seeds. Flag clearly.
  - **Cycle plot** — 12 mini-panels share a y-scale. The connective tissue is a monthly-mean horizontal within each panel; without it the seasonal separation doesn't read.
  - **Decision tree / Organizational chart** — both use tree layouts but have very different visual conventions. Decision tree: chance-node circles + decision-node squares + edge labels. Org chart: all rectangles, hierarchical positioning, no edge labels. Keep them visually distinct.
- **Slot routing:** Wave B has four layout-heavy slots (arc-diagram W, decision-tree L, organizational-chart L, timeline W). Route those to Opus. Everything else Sonnet.
