Story: leverage measures how far observation i is from the centre of X-space; it's a property of the regressors ALONE and doesn't require the response y. ## Notes

- **Deps likely needed:** none. All charts use SVG primitives + existing visx/d3 packages. Circos-plot reuses `d3-chord` (added Batch 2). Tree-of-life radial layout reuses `d3-hierarchy` (added Batch 1). Karyotype + ideogram draw G-bands as alternating dark/light rectangles — no banding library needed.
- **Known trickiness:**
  - **Cook's distance & leverage** stem-plots need the horizontal threshold line clearly drawn (4/n for Cook's, 2(p+1)/n for leverage) — not just labelled.
  - **Bagplot** halfspace-depth computation is O(n²) for the bag boundary; for n=60 inline that's ~3600 ops — acceptable. Use Rousseeuw's BAG algorithm or approximate with `d3-hull` + depth-peeling; simpler: pre-compute the 50% and outer-fence convex hulls deterministically and render them as polygons. Don't try to implement full Tukey depth at render time.
  - **Biplot** variable arrows need explicit loadings matrix pre-computed for the synthetic dataset. Generate 5 variables with a known correlation structure (seeded LCG) → compute first 2 PCs → plot observation scores + variable loading vectors. Hand-pick 5 variables + a correlation matrix so the biplot reads cleanly (some orthogonal, some collinear).
  - **Kagi / Renko / P&F / Heikin-Ashi** all need a synthetic price series as input. One seeded LCG random-walk serves all four — each chart interprets the same OHLC stream differently. Kagi: reversal threshold. Renko: brick size. P&F: box size + 3-box reversal. Heikin-Ashi: smoothing formula.
  - **Titration curve** — for strong-acid / strong-base the pH is piecewise computable: pH = 1 + log₁₀(V_NaOH / (V_initial + V_NaOH)) in the titrant-excess region, etc. Easier to hand-tabulate the curve's key points and `curveMonotoneX` interpolate between them. Avoid full equilibrium calculation at render.
  - **Arrhenius plot** x-axis is `1/T` — unusual inversion. Use `scaleLinear` on `1/T` values directly (pre-computed).
  - **UML use-case** stick-figure actors need proper SVG path (head circle + body+arms+legs lines). Hand-draw as a small reusable SVG group.
  - **UML component** lollipop-and-socket notation: lollipop = stick-line from component edge ending in a small circle (provided interface); socket = stick-line ending in a half-circle opening toward a lollipop (required interface). Components connect when a lollipop fits into a socket — draw them tangent.
  - **UML deployment** 3D-box nodes: render as an isometric-ish rectangle with a small diagonal tab at the top-right to hint at 3D depth. Don't try full 3D projection — the notation is iconic, not perspective-correct.
  - **UML object** instance rectangles have the colon-syntax label (`instanceName:ClassName`) UNDERLINED — use `text-decoration="underline"` on the `<text>` element.
  - **Circos plot** — chromosome ring = 22 adjacent arc segments. Inner data tracks = concentric rings at smaller radii. Ribbons = cubic Bezier paths through the chart centre connecting source-arc to target-arc. Use `d3-chord`-style arc path generators or hand-compute with polar→cartesian conversion.
  - **Tree of life radial** — `d3-hierarchy.cluster()` with `size([2*Math.PI, radius])` gives polar (angle, radius) coordinates for a radial tree. Convert to cartesian with `[r*cos(θ), r*sin(θ)]`.
  - **Pedigree chart** — mate lines are horizontal; parent-to-offspring lines drop vertically from the midpoint of the mate line and split into a horizontal "sibship" line just above the children row. Standard conventions critical: squares/circles have fixed size (not aspect-ratio adjusted).
  - **Karyotype** G-banding — each chromosome has ~15-30 bands. Don't hand-code all 22×46 band positions — use a simplified schematic: 4-6 bands per chromosome-pair with representative dark/light alternation. The chart is iconic, not a reference cytogenetic standard.
  - **Sequence logo** letter-sizing — each letter stretches vertically to fill its per-position allocation (info-content × frequency). Use `<text>` with `textLength` and `lengthAdjust="spacingAndGlyphs"` to force a fixed width per position while varying height. Or: hand-draw A/C/G/T as SVG paths for a reference-quality logo — simpler for a 4-letter alphabet.
  - **Ideogram** — one chromosome's worth of G-banding. Pick human chromosome 17 (gene-dense, contains BRCA1 and TP53 — the two annotation hooks). Centromere constriction drawn as a narrow waist.

  - **Wave B** — Opus for B1 (UML use-case actors + system-boundary + `<<include>>` relationships; UML component with lollipop-socket provided/required interfaces — distinct glyph vocabulary), B2 (UML deployment with 3D-box nodes containing artifact rectangles; UML object with underlined-instance-label syntax + link-labels — distinct rendering patterns), B3 (circos-plot with concentric ring tracks + inter-chromosomal ribbons — complex polar/path math; tree-of-life radial phylogeny with 30+ leaves at uniform angular spacing — layout-sensitive).
