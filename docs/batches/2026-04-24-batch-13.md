## Notes

- **Deps likely needed:** none. All 20 charts compose from existing SVG primitives + `d3-shape` + `d3-array` + `d3-hierarchy`.
- **Known trickiness:**
  - **Spiral-plot** — Archimedean r = a + bθ; sample 365 × 20 points at θ_i = 2π · i / 365. Convert (r, θ) → (x, y) for SVG. Encode value as stroke-width or fill colour on each segment (line built as a series of small segments, not one path).
  - **Comet-plot** — N=12 points tail. Each data point renders as (a) a head circle at opacity 1, size 5px; (b) a tapered tail — polyline through the last 12 positions with varying stroke-width and opacity (both decreasing linearly from head to tail). Use `stroke-dasharray` or draw as 12 individual segments.
  - **Shmoo-plot** — 16×16 grid. Each cell coloured green (pass) or red (fail). Boundary interpolated by testing `isFail(vcc, freq) = vcc < 1.0 + 0.002 * freq`. Draw with `<rect>` per cell. Add contour polyline on the boundary for emphasis.
  - **Carpet-plot** — Two families of intersecting curves. Hard-code an aerospace example: 5 gross-weight curves × 5 wing-loading curves. Each curve is a parametric line through 20 sample points. Label each curve's parameter value at its endpoint.
  - **Strip-chart** — Rolling ECG trace. Pre-compute one cycle of a synthetic QRS-complex (P-wave + QRS + T-wave bumps) and tile it 80× across the plot width. No real animation — render a static frozen moment with ~3-4 complete cycles visible; add small "recording" annotations (timestamp, speed "25 mm/s").
  - **Drain-plot** — Shockley's MOSFET equations: triode I_D = k[(V_GS-V_th)V_DS - V_DS²/2]; saturation I_D = k(V_GS-V_th)²/2. Plot 5 curves for V_GS ∈ {1.0, 1.5, 2.0, 2.5, 3.0} V over V_DS ∈ [0, 5] V. Annotate the "knee" where triode transitions to saturation on the V_GS=2.5 curve.
  - **Isarithmic-map** — Simplified US outline + 5 isohyet contour polygons at 10/20/30/40/60 in/yr. Don't compute real interpolation; hand-draw plausible closed polylines running north-south (Pacific Coast wet), west-east (Great Plains transition), then wrapping through the Southeast.
  - **Vertical-bar-graph** — Playfair's 1781 Scotland imports reproduction. Hard-code 17 countries and values from historical sources (or plausibly close: Ireland £117k, Russia £87k, Sweden £74k, Norway £55k, Denmark £45k, America (rebellious, so small) £23k, …, down to Flanders £5k). Alternate light/dark bar shading. No title decoration — label bars at their bases; values above each bar. **Differentiate from bar-chart by framing the narrative as historical Playfair reproduction.**
