## Notes

- **Deps likely needed:** none. Isosurface-plot's 3D mesh can be pre-computed into a hardcoded vertex list; marching-cubes need not run at render time.
- **Known trickiness:**
  - **Table-chart** — genuine table, not a chart with encoding. Row/column borders, cell padding, alternating row backgrounds. Make John Snow's 1854 Broad Street cholera data the body: house number, water source, deaths, "distance to pump". Include a bottom-row total and a "source" caption.
  - **Vowel-chart** — reversed axes: x = front→back (right), y = close→open (bottom). Trapezoidal quadrilateral frame mirrors IPA chart. Each vowel as small glyph with IPA symbol (use Unicode /i/, /ɛ/, /ɑ/, etc.).
  - **Dot-matrix** — rows of dots. Rendering: per-row `<circle>` or `<rect>` with fill/no-fill. Keep row labels left-side.
  - **Isotype** — repeated stick-figure path, scaled to fit ~10px wide. Rows label left-side; count displayed right-side.
  - **Icon-array** — 10×10 grid of stick figures. N "event" icons coloured dark (start from top-left); remaining 100-N coloured light grey. Below: "X in 100" + verbal frame ("X out of every 100 adults…").
